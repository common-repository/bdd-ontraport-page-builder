Ontraport Visual Builder integrates the Ontraport CRM with WP Bakery's Visual Builder. Try it now!

== Description ==
Utilising simple checkboxes, Ontraport Visual Builder allows you to easily select the page elements visible to specific Ontraport membership groups.

== Installation ==

*Server Requirements:* PHP4+

*Wordpress versions:* Wordpress 2.7 and up.

*Additional Plugins:* <a href="https://wordpress.org/plugins/pilotpress/">Ontraport PilotPress</a> and <a href="https://wpbakery.com/">WP Bakery Visual Builder</a>

*Note:* You require an Ontraport account : <a href="https://ontraport.com/">Create one here</a>.

Step-by-step Guide:

* Install plugin from WordPress directory and activate it.
* When editing pages simply edit the row or column and utilise the Ontraport tab.
